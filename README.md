# 📋 Prerequisites
```
JDK 1.8+ (推荐 azul-1.8.0_442 或更高版本)
Maven 3.3.9 或更高版本
netty 4.1.94.Final
```

# 🚀 快速开始

## 构建项目

**构建所有模块：**
```bash
mvn clean initialize

mvn package

mvn clean install
```
